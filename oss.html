EffectFactory

Python desktop tool for generating loopable black-background motion overlays for videos, streams, and other creator workflows.

  • Python
  • Tkinter
  • ffmpeg
  • MIT

Includes procedural effects, preview rendering, reproducible seeds, presets, and a plugin format for new effects.

GitHub

Chikyu-gata Historical Globe

Lightweight 3D historical globe for exploring representative eras, border data, and Earth-history notes in a local browser or Electron app.

  • Three.js
  • GeoJSON
  • Electron
  • MIT

Built for inspectable educational visualization with local data files, a timeline UI, and source notes for historical boundaries.
